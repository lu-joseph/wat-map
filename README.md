# wat-map
Map for all of University of Waterloo campus!

To run backend:

1. Navigate to `wat-map/backend/`

2. Launch MongoDB:

   `mongod`
  
3. In a new Linux terminal session, enter MongoDB Shell:

    `mongo`
  
4. Run API routes:

    `node dataserver.js`
  
5. Navigate to `wat-map/frontend/`

6. Host front end:

    `ng serve`
